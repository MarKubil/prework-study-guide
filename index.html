HTML

  • The head element contains information about the webpage.
  • The body element represents the visible content shown to the user.
  • The h1 elements have to be used once in a website followed by smaller heading elements.
  • The unordered list element ( <ul> ) shows bullet points on list elements ( <li>).
  • The ordered list elements ( <ol> ) shows numbers on list elements ( <li>).

CSS

  • A margin indicates how much space we want around the outside of an element.
  • A padding indicates how much space we want around the content inside an element.
  • A width and height indicates size.
  • A font-size indicates letter size.
  • A font-weight indicates letter thickness (bold, italic...)
  • A class selector could be reused and it's good practice to use it for DRY.
  • A id selector have to be crated unique and used only once.
  • There is 3 ways to use CSS inline, internal, external.

Git

  • GitHub hosting where kept all website files.
  • GitPage free static website domain.
  • Git command console.
  • Git some commands:
    • "git init" = initialize an existing directory as a Git repository
    • "git clone [url]" = retrieve an entire repository from a hosted location via URL
    • "git checkout" = switch branch
    • "git checkout -b [branch name] = create branch
    • "git status" = show modified files in working directory, staged for your next commit
    • "git pull origin main" = to pull main branch
    • "git push origin branch" = to push branch

JavaScript

  • Data types:
    • String = A string (or a text string) is a series of characters like "John Doe".
    • Numbers = JavaScript has only one type of numbers.
    • Booleans = Booleans can only have two values: true or false.
    • Arrays = JavaScript arrays are written with square brackets. Array items are separated by commas.
    • Objects = JavaScript objects are written with curly braces {}. Object properties are written as name:value pairs, separated by commas.
  • JavaScript is case sensitive.
  • The if/else statement executes a block of code if a specified condition is true.
  • for loop is a JavaScript for loop executes a block of code as long as a specified condition is true.
  • Function is simply a set of instructions that tells the computer how to perform a certain task.
